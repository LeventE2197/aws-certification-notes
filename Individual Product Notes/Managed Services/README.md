## A Cloud Guru
该服务不出现在控制台里（通过直接联系 AWS 销售人员来开通，一般提供给较大的企业的服务），是 AWS 提供管理开发者的 AWS 基础设施的的 ongoing 管理服务，因而开发者只需专注开发即可。Manage Services 会根据最佳实践来实现、维护开发者的基础设施，包括：  
* Change Management
* Incident Management
* Provisioning Management
* Patch Management
* Access Management
* Security Management
* Continuity Management
* Reporting
* ITSM Integration
